팀 최우선 규약 — UI Kit First + UIUX 규칙 준수 (3축)
① 모든 화면 UI/UX 생성 시 본 UI Kit 을 최우선 참고원으로 사용한다.
② UI Kit 에 없거나 신규로 구성되는 UI/Component 는 반드시 UI Kit 에 먼저 등록한 뒤 화면에 적용한다.
