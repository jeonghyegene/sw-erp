성원 애드피아 MES/ERP 시스템 UIUX 공통 가이드



1. 날짜 및 시간 표시 규격

1.1 일자 표시

형식: YY/MM/DD
예시: 25/02/09
적용: 모든 날짜 필드, 그리드, 조회 조건

1.2 일시 표시

형식: YY/MM/DD   HH:MM (공백 3칸)
예시: 25/02/09   14:30
적용: 등록/수정 일시, 작업 시간 기록



2. 금액 표시 규격

2.1 금액 정렬

그리드 정렬: 우측 정렬
적용: 모든 금액 컬럼

2.2 금액 표기

형식: 천 단위 구분 (콤마)
예시: 1,234,500
적용: 모든 금액 표시 영역



3. 그리드(테이블) 표시 규격

3.1 컬럼 정렬 기준

좌측 정렬: 텍스트, 명칭, ID, 이메일, 이름
중앙 정렬: 체크박스, 아이콘, 버튼, 상태, 코드, 날짜, 유형
우측 정렬: 숫자, 금액, 수량

3.2 행 높이 및 밀도

기본 행 높이: 37px
컴팩트 모드: 37px (대량 데이터 조회 시)
여유 모드: 48px (입력 중심 화면)

3.3 페이징

기본 페이지 크기: 20건
선택 옵션: 20 / 50 / 100 / 200 / 500 / 1,000
표시 정보: "1-20 / 총 156건"
페이징: \[〈/〉] 이전/다음 1페이지씩 이동, \[《/》] 맨처음/맨끝 페이지 이동



4. 입력 폼(Form) 규격

4.1 필수 입력 표시

표시 방법: 라벨 우측에 \* 빨간색 표시
예시: 주문번호 \*
안내 문구: 필수 입력 항목입니다

4.2 입력 필드 크기

짧은 입력: 120px (코드, 수량 등)
중간 입력: 240px (이름, 품목 등)
긴 입력: 480px (주소, 비고 등)
전체 너비: 부모 컨테이너 100%

4.3 입력 도움 요소

Placeholder: 입력 예시 표시 (회색 텍스트)
Helper Text: 필드 하단에 안내 문구
Tooltip: 라벨 옆 ? 아이콘 hover 시 설명

4.4 입력 제약 및 검증

유효성 체크: 최종 저장 및 수정 이벤트 등록 버튼 클릭 시
오류 표시: 필드 테두리 빨간색 + 하단 오류 메시지



5. 버튼 규격

5.1 버튼 우선순위

Primary: 주요 액션 (저장, 등록, 확인) - 파란색
Secondary: 보조 액션 (취소, 닫기) - 회색 테두리
Danger: 삭제, 취소 등 주의 필요 - 빨간색
Text: 부가 기능 (상세보기, 더보기) - 텍스트만

5.2 버튼 크기

대: 32px 높이 (주요 액션)
중: 28px 높이 (기본)
소: 24px 높이 (그리드 내부)

5.3 버튼 배치

폼 하단: 우측 정렬 (취소 - 저장 순서)
그리드 상단: 좌측 정렬 (조회 - 등록 - 출력 순서)
행 액션: 그리드 우측 고정 컬럼



6. 상태 및 알림 표시 규격

6.1 상태 뱃지

진행 중: 파란색 뱃지
완료: 초록색 뱃지
대기: 회색 뱃지
취소/오류: 빨간색 뱃지

6.2 토스트 메시지

성공: 우측 하단, 초록색, 3초 자동 닫힘
오류: 우측 하단, 빨간색, 수동 닫기
정보: 우측 하단, 파란색, 5초 자동 닫힘

6.3 확인 다이얼로그

제목: 명확한 액션 설명
내용: 2-3줄 이내 간결한 설명
버튼: 취소(좌측) - 확인(우측)



7. 검색 및 필터 규격

7.1 날짜 범위 검색

날짜 입력: 시작일/종료일 분리된 필드 (YYYY-MM-DD 형식)
기간 탭: 오늘, 1주일, 1개월, 3개월, 6개월, 1년 (Quick Filter)
날짜 유형 선택: 드롭다운으로 가입일/주문일/출고일 등 선택(조건부)



7.2 검색 영역 구성

기본 검색: 드롭다운으로 검색조건 설정 + 검색어 입력 + 검색 버튼
상세 검색: 접기/펼치기 토글
검색 조건: 최대 10행까지 표시

7.3 상세 검색 영역

그룹화된 필터: 진행현황, 결제수단, 접수경로 등 카테고리별 구분
체크박스 필터: 수평 배치, "전체" 옵션 포함
라디오 버튼: 단일 선택 옵션
텍스트 입력: "공미(,) 구분 입력" 안내
드롭다운: 선택형 필터

7.4 자동완성 검색

트리거: 2글자 이상 입력 시
결과 표시: 최대 10개
하이라이트: 입력 문자 강조 표시

7.5 검색 설정 저장 기능

저장: ☆ 검색설정 저장 버튼
불러오기: ★ 검색설정 1, 2, 3... 태그 형식
관리: × 버튼으로 삭제, 최대 3개 저장



8. 코드 및 식별자 표시 규격

8.1 코드 표시 형식

표시 방법: 코드(블루컬러+언더바) 명칭
예시: C001(블루컬러+언더바) 일반 고객
구분자: 블루컬러+언더바 적용

8.2 주요 코드 체계

고객 코드: \[기업 CA~CZ + 숫자 5자리], \[개인 GA~GZ + 숫자 5자리], \[외국인 FA~FZ + 숫자 5자리]
품목 코드: AA~ZZ + 숫자 3자리
원자재 코드: AA~ZZ + 숫자 2자리
부자재 코드: AA~ZZ + 숫자 2자리+ 숫자 2자리
장비코드: AA~ZZ + 숫자 2자리+ 숫자 2자리
부품코드: AA~ZZ + AA~ZZ + 숫자 2자리
주문 번호: SW+년도(YY) + 순번 5자리
작업 지시 번호: W + 날짜(YYMMDD) + 순번 3자리

9. 권한 및 비활성화 표시 규격

9.1 비활성화 요소

시각적 표시: 투명도 50%, 커서 not-allowed
텍스트 색상: 회색 (#999999)
툴팁 안내: "권한이 없습니다" 또는 "조건 미충족"

9.2 읽기 전용 필드

배경색: 연한 회색 (#F5F5F5)
테두리: 점선 또는 없음
아이콘: 잠금 아이콘 표시 (선택적)



10. 파일 첨부 규격

10.1 파일 업로드

드래그 앤 드롭: 지원
파일 선택: 버튼 클릭
다중 선택: 가능
최대 크기: 10MB per file
허용 확장자: PDF, JPG, PNG, XLSX, DOCX

10.2 첨부 파일 표시

파일명: 30자 초과 시 말줄임(...) / 호버시 전체 파일명 툴팁 표시
파일 크기: KB/MB 단위 표시
다운로드: 파일명 클릭
삭제: X 버튼 (업로드 전/권한 있을 때만)



11. 로딩 및 대기 상태 규격

11.1 로딩 인디케이터

전체 화면: 중앙 스피너 + "로딩 중..." 문구
부분 영역: 해당 컴포넌트 위 오버레이
버튼 내부: 버튼 비활성화 + 작은 스피너

11.2 대량 데이터 처리

진행률 표시: 프로그레스 바 + "150/500 처리 중"
취소 옵션: 취소 버튼 제공
완료 알림: 토스트 메시지 + 결과 요약



12. 포커스 표시

12.1 입력 필드 포커스

파란색 2px 테두리
0.2초 부드러운 전환 효과
일반/포커스 상태 시각적 예시

12.2 그리드 행 포커스/선택

Hover: 연한 회색 배경
선택: 연한 파란색 배경
포커스: 파란색 테두리

12.3 다중 선택 시 포커스

체크박스 선택 시 배경색
선택 개수 표시
ESC 키로 일괄 해제



13. 오류 처리 및 빈 상태 규격

13.1 빈 상태 (Empty State)

아이콘: 회색 일러스트
메시지: "조회된 데이터가 없습니다"
액션: "새로 등록하기" 버튼 (권한 있을 때)

13.2 오류 페이지

404: "페이지를 찾을 수 없습니다"
403: "접근 권한이 없습니다"
500: "일시적인 오류가 발생했습니다"
액션: "이전 페이지로" 버튼

14. 승인/결재 플로우 표시 규격

14.1 승인 상태 시각화

대기: 회색 원형 아이콘
진행 중: 파란색 원형 아이콘 (애니메이션)
승인: 초록색 체크 아이콘
반려: 빨간색 X 아이콘
표시 형식: 단계명 + 승인자명 + 일시

14.2 승인선 표시

레이아웃: 좌→우 타임라인 형식
연결선: 점선 또는 화살표
현재 단계: 강조 표시 (굵은 테두리)
예시: 작성자 → 팀장(승인) → 부서장(대기) → 임원(미진행)

14.3 승인 이력

표시 항목: 승인자, 승인일시, 승인의견, 첨부파일
정렬: 최신순 (최근 승인이 상단)
의견 입력: 최대 500자, 필수/선택 구분



15. 알림(Notification) 규격

15.1 알림 유형

시스템 알림: 작업 완료, 배치 처리 결과
업무 알림: 승인 요청, 납기 임박, 재고 부족
개인 알림: 멘션, 댓글, 공지사항

15.2 알림 표시 위치

헤더 벨 아이콘: 읽지 않은 알림 개수 뱃지 (빨간색 숫자)
알림 패널: 벨 아이콘 클릭 시 우측 슬라이드 패널
패널 높이: 최대 400px, 스크롤 가능

15.3 알림 항목 구성

제목: 굵은 글씨, 최대 50자
내용: 일반 글씨, 최대 100자
시간: "3분 전", "2시간 전", "2일 전" 상대 시간 표시
액션: "바로가기" 링크 (해당 화면으로 이동)
읽음 처리: 클릭 시 자동, 일괄 읽음 처리 버튼 제공



16. 이력/로그 조회 규격

16.1 변경 이력 표시

표시 항목: 변경일시, 변경자, 변경 전 값, 변경 후 값, 변경 사유
색상 구분: 변경 항목 하이라이트 (노란색 배경)
비교 모드: 이전 버전과 현재 버전 좌우 비교

16.2 작업 로그

로그 레벨: INFO(파란색), WARNING(주황색), ERROR(빨간색)
타임스탬프: YY/MM/DD HH:MM (초 단위 미포함)
필터링: 날짜 범위, 사용자, 액션 유형
내보내기: CSV, Excel 다운로드 지원

16.3 감사 추적 (Audit Trail)

추적 대상: 중요 데이터 생성/수정/삭제
저장 항목: IP 주소, 사용자 ID, 액션, 이전/이후 데이터
보관 기간: 최소 3년
접근 권한: 관리자 또는 감사 담당자만



17. 다중 선택 및 일괄 처리 규격

17.1 다중 선택 방식

전체 선택: 헤더 체크박스
부분 선택: 각 행 좌측 체크박스
범위 선택: Shift + 클릭 지원
선택 상태 표시: 선택된 행 배경색 변경 (연한 파란색)

17.2 선택 상태 안내

선택 개수 표시: "15개 항목 선택됨"
위치: 그리드 상단 좌측
선택 해제: "선택한 체크박스 개별 해제"

17.3 일괄 액션

액션 버튼: 그리드 컬럼 우측 배치
제공 액션: 일괄 삭제, 일괄 승인, 일괄 다운로드, 일괄 출력
확인 프로세스: "15개 항목을 삭제하시겠습니까?" 다이얼로그



18. 탭(Tab) 네비게이션 규격

18.1 탭 스타일

Pill 스타일: 둥근 모서리 (20-24px)
활성 탭: 파란 배경 + 흰 글씨
비활성 탭: 회색 글씨, 투명 배경
Hover: 연한 파란색/회색
언더라인 스타일: 대안 디자인 제공

18.2 탭 내용 전환

페이드 인/아웃 애니메이션 (0.2-0.3초)
로딩 처리 (스피너 + 메시지)
URL 파라미터 반영 (북마크/뒤로가기 지원)
기본 탭: 첫 번째 탭

18.3 탭 뱃지

위치 및 형태 (원형/둥근 사각형)
알림 뱃지: 탭 이름 우측 빨간색 원형 숫자
예시: "승인 대기 (3)"
업데이트: 실시간 또는 페이지 로드 시



19. 트리 구조 및 계층 표시 규격

19.1 트리 네비게이션

펼침/접기: 폴더 아이콘 클릭 또는 ▶/▼ 화살표
들여쓰기: 레벨당 24px
최대 깊이: 5단계 (성능 고려)
아이콘: 폴더(접힌 상태), 열린 폴더, 파일

19.2 계층 구조 표시

BOM(자재 명세서): 부모-자식 관계 들여쓰기
조직도: 트리 또는 조직도 차트 형식
카테고리: 대분류 > 중분류 > 소분류 (> 구분자)

19.3 트리 액션

전체 펼치기/접기: 상단 토글 버튼
노드 검색: 검색어 입력 시 해당 노드 자동 펼침 + 하이라이트
드래그 앤 드롭: 순서 변경 또는 상위 카테고리 변경 (권한 필요)



20. 도움말 및 가이드 규격

20.1 인라인 도움말

Tooltip: 라벨 옆 ? 아이콘, hover 시 설명 (최대 200자)
Helper Text: 입력 필드 하단 회색 작은 글씨
예시 텍스트: Placeholder 활용

20.2 화면 단위 도움말

위치: 화면 우측 상단 ? 버튼
형식: 사이드 패널 또는 모달
내용: 화면 목적, 주요 기능, 사용 방법, FAQ

20.3 튜토리얼/온보딩

신규 사용자: 첫 로그인 시 주요 화면 투어 (Skip 가능)
신규 기능: 기능 출시 시 "New" 뱃지 + 간단한 설명
단계별 가이드: 복잡한 프로세스는 1단계, 2단계... 안내



21. 즐겨찾기 및 최근 사용 규격

21.1 즐겨찾기 메뉴

등록 방법: 메뉴 우측 별(☆) 아이콘 클릭
해제 방법: 별(★) 아이콘 다시 클릭
표시 위치: 상단 네비게이션 또는 좌측 메뉴 상단
최대 개수: 10개

21.2 최근 사용 메뉴

자동 기록: 메뉴 클릭 시 자동 저장
표시 개수: 제한 없음
표시 위치: GNB(헤더) 하단 탭 메뉴 구성
표시 형식: 메뉴명



22. 엑셀 업로드/다운로드 규격

22.1 엑셀 업로드

템플릿 제공: "템플릿 다운로드" 버튼
파일 선택: "파일 선택" 또는 드래그 앤 드롭
검증 단계: 업로드 전 데이터 유효성 검증
오류 처리: 오류 행 번호 + 오류 내용 표시, 오류 파일 다운로드

22.2 엑셀 다운로드

다운로드 범위: 현재 페이지 / 전체 / 선택 항목
파일명: 화면명\_날짜.xlsx (예: 주문목록\_250209.xlsx)
시트 구성: 데이터 시트 + 메타정보 시트 (다운로드 일시, 조건 등)
대용량 처리: 10,000건 이상 시 백그라운드 처리 + 알림
개인정보 마스킹: 개인정보 포함 시 마스킹 처리여부 안내 후 처리/미처리 선택에 따라 마스킹 처리
**민감정보: 이름, 휴대폰번호, 주소, 주민번호, 운전면허증번호, 여권번호, 외국인등록번호
-> 안내문구: 개인정보 마스킹 처리 후 다운로드 하시겠습니까? (홍*동, 010-****-5678)
-> 버튼: (Primary: 마스킹 후 다운 / Secondary: 마스킹 없이 다운)

22.3 엑셀 데이터 형식

날짜: 텍스트 형식 (YY/MM/DD) - 엑셀 자동 변환 방지
숫자: 숫자 형식, 천 단위 구분
긴 숫자: 텍스트 형식 (사업자번호, 계좌번호 등)



23. 권한별 UI 차별화 규격

23.1 메뉴 표시

권한 없음: 메뉴 자체 미표시
읽기 권한: 메뉴 표시, 조회만 가능
쓰기 권한: 메뉴 표시, 등록/수정 가능
삭제 권한: 삭제 버튼 활성화
콘텐츠 노출: 노출 활성화 콘텐츠만 표시
버튼 노출: 노출 활성화 버튼만 표시
마스킹: 권한 설정에 따라 콘텐츠 부분/전체 마스킹 표시

23.2 필드 편집 권한

읽기 전용: Read Only
조건부 편집: 권한설정에 따라 편집 가능 (예: 수정가능)
전체 편집: 일반 입력 필드

23.3 수평 스크롤

고정 컬럼: 컬럼 고정 좌측(1~2개)설정
스크롤 힌트: 좌우 그라데이션
스크롤 위치: 스크롤바



24. Breadcrumb (경로 표시)

표시 위치: 화면 상단 우측
형식: 홈 > 주문 관리 > 주문 목록
클릭: 각 단계로 이동 가능
현재 페이지: 굵은 글씨, 클릭 불가



25. 뒤로가기 처리

브라우저 뒤로가기: 이전 화면으로 복귀, 조회 조건 유지
취소 버튼: 변경 사항 미저장 시 확인 다이얼로그



26. 주석 및 메모 기능 규격

26.1 항목별 메모

표시: 메모 아이콘 + 메모 개수
작성: 말풍선 형식, 작성자/작성일시 표시
편집/삭제: 본인 작성 메모만 가능

26.2 전체 메모 영역

위치: 화면 중앙 모달(Height: max 700px)
내용: 기본 100자 이상 더보기 버튼(접기/펼치기 아코디언 기능)
정렬: 최신순/오래된순



27. 설정 및 환경설정 규격

27.1 사용자 환경설정

언어: 한국어

개인 설정: 사용자별 컬럼 표시 저장
