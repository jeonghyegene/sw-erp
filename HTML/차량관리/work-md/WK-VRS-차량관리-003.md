## 1) Summary

### One-liner

전체 차량의 현황·정비이력·운행이력을 관리하고, 정비이력 등록 및 상세 조회를 수행하는 관리자/사용자 공용 화면

### Goals

- 목표 1: 차량 현황(전체/가용/운행 중/미확인 예외)을 대시보드로 파악
- 목표 2: 차량별 정비이력과 운행이력을 풀팝업으로 상세 조회
- 목표 3: 정비이력을 신규 등록하고 첨부파일 포함 상세 확인

### Scope (In)

- ✅ 차량 현황 요약 카드 (4종 통계)
- ✅ 차량 목록 테이블 (차량사진, 차량명, 차량번호, 누적 거리, 정비이력, 누적 정비 금액, 현재 상태, 운행상태, 운행이력)
- ✅ 정비이력 풀팝업 (차량별 정비 내역 리스트 + 페이지네이션)
- ✅ 정비이력 상세 OffCanvas (정비 정보 카드 + 첨부파일 이미지 + 좌측 확장 패널)
- ✅ 운행이력 풀팝업 (차량별 운행 내역 리스트 + 페이지네이션)
- ✅ 운행이력 상세 OffCanvas (운행 거리 카드 + 운행전/후 사진 카드 + 진행이력 카드)
- ✅ 정비이력 등록 모달
- ✅ 운행상태 관리 (운행가능/운행불가/정비중 드롭다운, 예약 가용성 연동)
- ✅ 차량관리 상세 OffCanvas에서는 운행종료·예약취소 버튼 미노출 (내 예약 목록 전용)

### OffCanvas 상세 — 정비이력

```
┌──────────────────────────┐
│ 정비 상세 정보        [X] │  ← offcanvas-header
├──────────────────────────┤
│ [정비 정보 카드]          │  ← oc-card (flex-shrink:0)
│  차량명, 정비유형, 기간   │
│  정비소, 비용, 설명       │
├──────────────────────────┤
│ [첨부파일 카드]           │  ← oc-card
│  섬네일 그리드            │
│  클릭 → 좌측 확장 패널    │
└──────────────────────────┘
```

### OffCanvas 상세 — 운행이력

```
┌──────────────────────────┐
│ 운행 상세 정보        [X] │
├──────────────────────────┤
│ [운행 거리 정보 카드]     │  ← 출발/도착 거리, 운행 거리, 갭
├──────────────────────────┤
│ [운행전/후 사진 카드]     │  ← 사진 그리드, hover → lightbox
├──────────────────────────┤
│ [진행이력 카드]           │  ← 타임라인(예약→운행→종료→반납)
└──────────────────────────┘
```
