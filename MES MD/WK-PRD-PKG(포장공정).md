## 1) Summary

**One-liner**: 가공 완료된 품목을 바코드 스캔으로 조회하고, 측정무게·라벨 수량 입력 후 포장 검증 모달(Double Check)을 거쳐 라벨·송장을 자동 출력하며 포장 완료 → 출고대기 자동 전환까지 처리한다.

### Goals

- 바코드 스캔 한 번으로 포장 대상 작업의 모든 정보(수량·배송지·가공옵션·메모)를 한 화면에서 확인한다
- 측정무게 입력 → 포장 검증 모달 → 라벨·송장 자동 출력으로 오출고 없이 포장 공정을 완료한다
- 금일 완료 이력을 좌측 사이드바에서 실시간 조회하고 완료 건의 상태 되돌리기(출고 전 한정)를 지원한다
- 자재 부족 시 MES 포장 화면 내에서 창고 담당자에게 자재 신청·취소를 처리한다

### Scope

**In Scope**
- 포장 기기 선택 (포장 1호기 / 2호기, 수동 선택)
- 바코드 스캔 기반 포장 대상 작업 조회 (스캔 키: 품목번호)
- 포장 상세 정보 표시 (접수번호·고객·수량(정매/여분)·배송지·배송유형·용지·가공옵션)
- 고객 요청사항·관리자 메모 표시
- 측정무게 입력(kg, 소수점 1자리), 라벨 수량 입력
- 포장 검증 모달 (Double Check, 강제 체크박스 없음)
- 포장 완료 처리 → 라벨/송장 자동 출력 → 출고대기 자동 전환
- 포장 완료 취소 (출고 전 한정, 역전이 허용)
- **일자별 이력 사이드바 조회** (상단 날짜 선택 + 페이지네이션, v1.4.2 개정. 기본값: 오늘, 최근 완료 순)
- 자재 검색·신청·신청 취소 (창고 담당자 승인)
- 감사 로그 기록 (POL-MES-001을 따른다)
- 직접수령 배송유형 수령 장소 표시 (성수동/하남/충무로)
- 작업중지 처리 (포장대기/포장중 → 작업중지, 사유 기록, 재개 시 직전 상태 복귀)
- 어드민 포장내역 일괄 상태 변경과의 양방향 실시간 동기화
- 포장 완료 취소 직후 해당 품목 자동 재로딩 또는 이력 상세에서 '다시 포장' 버튼으로 재진입 (v1.4.0 신설, DEC-25)
- 상세뷰 좌측 상단 제품 이미지 대영역 노출 — 포장물 오인 방지 (v1.4.0 신설, DEC-26)

**Out of Scope**
- 작업 삭제 (POL-MES-005를 따른다. 상태 되돌리기만 허용)
- 출고·배송 관리 (MES 범위 외)
- 감리 승인 처리 (WK-PRD-AUD에서 담당)
- 자재 마스터 관리·재고 관리 (자재 도메인 담당)
- 포장 기기 마스터 등록 (시스템-코드관리 담당)
