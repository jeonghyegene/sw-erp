## 1) Summary

**One-liner**: 생산 현장의 모든 기기 상태를 실시간 카드·기간별 그리드·Gantt 타임라인 3가지 뷰로 모니터링하여 기기 가동률과 생산 진행 상황을 가시화하는 조회 전용 화면이다.

### Goals

- 기기 중단을 즉시 인지하고 상단 Alert 배너로 시각 알림을 제공한다
- 실시간 기기 카드로 KPI 6종(통/작업건수/정매+여분/최종가동/비가동/속도)을 한 화면에서 파악한다
- 기간별 기기 타입별 생산 매수를 최대 6개월 단위로 집계 조회하여 실적 분석을 지원한다
- Gantt 타임라인에서 기기별 작업 배분 현황을 시간 축으로 시각화하고 긴급·당일 작업을 식별한다

### Scope

**In Scope**
- MON-01 인쇄기기 모니터링: 실시간 기기 상태 카드, KPI, 중단 Alert 배너, 차트 영역
- MON-02 생산기기 가동현황: 기간별 기기 타입별 생산 매수 그리드, 일자별 라인 차트, 엑셀 다운로드
- MON-03 생산기기 타임라인: Gantt 바 시각화, Detail Drawer, 기기 정보 모달, 줌 레벨 조작
- 기기 상태 전이 관찰 (가동중/종료/고장), 중단 알림 이력 표시
- 엑셀 다운로드 감사 로그 기록 (POL-ADM-031 / POL-DATA-004)
- WebSocket 실시간 Push 수신 + 장애 시 HTTP 폴링 Fallback 전환

**Out of Scope**
- 기기 상태 수동 보정·작업 배정 변경 → WK-PRD-PRC, WK-PRD-FAB
- 기기 마스터 등록·수정 → 기준정보 모듈 (P_MES_ADMIN 전용)
- 알림톡/SMS 발송 → WK-PRD-AUD(감리)
- 생산 계획 수립 → 별도 모듈

| LR-040 | **MON-01 표준시간 경과율** — 기기 카드 KPI 영역(또는 작업 행 우측)에 표준시간 경과율(%)을 표시한다. 계산: 진행 중 = (현재시각 − `root.StartDate`) ÷ `ProcessFx.StandardTime` × 100, 완료 = `root.WorkingTime` ÷ `ProcessFx.StandardTime` × 100. 모든 시간 항은 분 단위로 통일 후 계산한다. 시그널 색상은 0~80% 정상(회색/초록), 80~100% 주의(노랑), 100~120% 경고(주황), 120%+ 위험(빨강)으로 표시한다. `StandardTime`이 0 또는 null이면 "—"를 표시한다.
