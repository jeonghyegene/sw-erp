## 1) Summary

### One-liner
> MES 전 공정에서 공유하는 기반 기능(3가지 로그인 방식, 홈 대시보드, 알림)과 공통 정책(스캔 원칙, 상태 모델, 배정 정책, 자재신청, 감리일정)을 정의한다.

### Goals

| Goal ID | Goal |
|---------|------|
| G-01 | 현장 환경에 맞는 로그인 방식 제공 (기기PC 클릭 / 사번 / NFC) |
| G-02 | 로그인 후 담당 공정/기기 현황을 즉시 파악할 수 있는 홈 구성 |
| G-03 | 알림을 통해 현장 작업자가 중요 정보를 놓치지 않도록 지원 |
| G-04 | 일관된 상태 관리와 작업 로그 기록 보장 |

### Scope

#### In-Scope

| # | 항목 | 설명 |
|---|------|------|
| 1 | 기기PC 담당자 선택 로그인 | 기기 옆 PC에서 IP 기반 담당자 목록 → 클릭 로그인 |
| 2 | 사번/비밀번호 로그인 | 일반 로그인 (사번 저장 옵션 포함) |
| 3 | NFC 태그 로그인 | NFC 사원증 태깅 → 즉시 로그인 |
| 4 | 홈 대시보드 | 공정별 탭(생산기기/가공기기) + 위치 필터 |
| 5 | 알림 | 생산/메시지/공지 3가지 카테고리, 미확인 건수 표시 |
| 6 | 공지사항 조회 | 그룹웨어 공지 중 MES 필터 (홈에서 접근) |
| 7 | 캘린더 조회 | 감리/전사/휴가 일정 통합 조회 |
| 8 | 감리일정 설정 | 접수 요청건에 대해 일정 설정 (관리자) |
| 9 | 자재신청 | 신청만 (승인/출고는 그룹웨어) |
| 10 | 마스터 데이터 조회 | 장비/사용자/코드 조회 (관리자) |
| 11 | 스캔/상태변경 공통 정책 | 확인 1회 정책, 상태 모델 |
| 12 | 감사 로그 | POL-MES-001을 따른다 |
| 13 | 자동 로그아웃 | 비활성 1시간 경과 시 자동 세션 종료 (OI-057) |
| 14 | 중복 로그인 방지 | 동일 사번 중복 로그인 불가, 이전 기기 강제 세션 종료 |
| 15 | 비밀번호 복잡도 정책 | 숫자+영문 조합 필수, 유효기간 없음 (OI-058) |
| 16 | GNB 공통 네비게이션 | 전체 MES 메뉴 구조, 로고, 메뉴 항목, 알림 아이콘, 프로필 드롭다운 |
| 17 | NFC 로그인 전용 화면 | NFC 태깅 영역, 스캔 결과 피드백, 대기 상태, 사번/비밀번호 전환 링크 |
