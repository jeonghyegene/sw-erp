## 1) Summary

**One-liner**: 생산관리에서 등록된 재작업 건을 **유발 주체(작업자/부서/외주업체) × 월(1~12)** 축으로 **건수·재작업액** 집계하여, MES 작업자·관리자가 현 작업과 무관하게 독립 윈도우 팝업으로 조회하는 **조회 전용** 화면이다.

### Goals

- 유발 주체별 월간 재작업 건수·재작업액을 한 화면에서 파악한다
- 행 클릭으로 해당 주체×월의 **접수번호 단위 재작업 사유·금액**을 즉시 확인한다
- 현재 작업과 분리된 독립 팝업으로 병행 조회를 지원한다
- 일반 사용자는 본인(유발 주체) 실적만, 마스터 관리자는 전체를 조회한다

### Scope

**In Scope**
- RWK-01 재작업현황 그리드 (고정 컬럼 + 월별 건수·재작업액 + sticky 합계행)
- RWK-02 재작업 상세 모달 (행 클릭 → 접수번호 단위 주문정보·사유·재작업금액)
- 필터 7종 (연도/담당선택/부서별 검색/부서선택(단일)/초기화/검색/엑셀출력)
- 권한 기반 행 수준 필터 (본인 / 마스터 전체)
- 엑셀 다운로드 + 감사 로그 (POL-ADM-031 / POL-DATA-004)
- 독립 윈도우 팝업 호출 (window.open, GNB·LNB 비노출)

**Out of Scope**
- 재작업 발생·등록·금액 수동 입력 → 생산관리 재작업 처리 화면 [교차참조: PROD / WK-ORD-RWK]
- 재작업 집계 산식·계수 기준·발생률·사유 마스터 정의 → 생산관리 재작업통계 [교차참조: PROD / WK-STA-RWK]
- 재작업 취소·환불·자동 처리 → [교차참조: PROD / WK-ORD-RWH(재작업내역)]
- 사유 1차/2차/3차 코드체계 정의 → [교차참조: PROD / WK-MST-RWR(재작업사유마스터)]
