## 1. 확정 마스터 값

### 1-1. 생산 공정 단계 (순서)

```
조판 → 출력/인쇄 → 후가공/재단 → 포장 → 출고
```

### 1-2. 조판 공정 세부 단계 (순서) — v1.1 신규

```
[수동조판] 조판 가져오기 → 조판 중 → 조판완료 → 기기 자동배정(배정완료) → 검수/배정확정 → 출력대기
[자동조판] 응용프로그램 실행 → 조판완료 → 기기 자동배정(배정완료) → 검수/배정확정 → 출력대기
```

### 1-3. Prepress 프로세스 단계 (순서)

```
RIP (포맷변환/CMYK분리/각도포함) → CIP (인쇄가능파일 변환) → CTP (인쇄판)
```

### 2-19. 용지 평량 및 띠지 무게 산정 — v1.11 신규

> 배송 라벨(띠지)에 표시되는 출력물 무게의 SoT. 평량 마스터·산정 공식·표시 규칙·예외 처리를 본 정책이 단일 결정 지점으로 관리한다. 띠지·통계·택배비 자동 세팅 등 모든 무게 파생 데이터는 본 정책 공식을 통해서만 산출된다.

| Rule ID | 내용 | 근거 |
|---------|------|------|
| POL-PRD-190 | 용지의 평량(weight_gsm, **단위: g/m²**)은 PaperMaster 마스터 필드로 관리한다. 평량 값의 등록·수정 SoT는 용지 마스터 관리 업무(WK-PRD-PAPER §9 PaperMaster)에 한정한다. 평량은 **필수 입력**이며, **음수·0은 허용하지 않는다**. 동일 용지군의 등급별 평량(예: 스노우지 80/100/120/150g)은 각각 별도 PaperMaster 레코드로 등재한다 | 사용자 결정 2026-05-27 |
| POL-PRD-191 | 단일 인쇄물의 총 중량(g) 산정 공식은 다음과 같다. **중량(g) = 평량(g/m²) × 가로(mm) ÷ 1000 × 세로(mm) ÷ 1000 × 매수**. 가로·세로는 접수건의 width_mm·height_mm 입력값을 m로 환산한 값이다. 표준 사이즈 코드(A4 등)를 사용하는 경우에도 mm 값으로 환산하여 동일 공식을 적용한다. **계산 예**: A4(210×297mm) 스노우지 150g/m² 1,000매 → 150 × 0.21 × 0.297 × 1,000 = **9,355.5g ≈ 9.36kg** | 사용자 결정 2026-05-27 |
| POL-PRD-192 | 1개 접수건이 복수 용지(예: 책자의 표지+내지, 다부속 제품)로 구성된 경우, 부속별 중량을 각각 POL-PRD-191 공식으로 산정한 뒤 **단순 합산**한다. **가공 자재(코팅·라미넥스·박 등)의 추가 중량은 본 공식 범위에 포함하지 않는다** — 가공 자재 중량 산정은 별도 정책에서 다루며, 현재 미정의([NEED-CONFIRM]) | 사용자 결정 2026-05-27 |
| POL-PRD-193 | 무게의 **내부 저장·계산은 g 단위(실수)**로 수행하고, **외부 표시(띠지·UI·통계)는 kg 단위**로 환산한다. 표시 규칙은 **소수 셋째자리에서 반올림한 소수 둘째자리까지** 노출한다(예: 9,355.5g → 9.36kg, 9,354.4g → 9.35kg). 표시 규칙의 최종 권위는 POL-DLV-050이며, 본 정책은 그 위임 결과로 산정값을 제공한다 | 사용자 결정 2026-05-27, POL-DLV-050 |
| POL-PRD-194 | 평량(weight_gsm)·가로(width_mm)·세로(height_mm)·매수 중 **어느 하나라도 누락·0·음수**인 경우, 본 공식은 중량을 산출하지 않는다. 띠지·UI에는 **"미산정"**(또는 빈 칸 + 안내 아이콘)으로 표시하고, **임의 기본값 대체·평균값 추정·인접 용지값 차용은 모두 금지**한다. 누락 보완은 각 데이터의 SoT 업무(평량 → WK-PRD-PAPER, 사이즈·매수 → 접수 SoT)에서 수행한 후 재계산을 트리거한다 | 사용자 결정 2026-05-27 |

> ※ 참조 업무: WK-PRD-PAPER §9 PaperMaster.weight_gsm (마스터 SoT), WK-DLV-MGT AC-011 Print_Label (소비측 — 띠지 출력), POLICY_BASE_DELIVERY POL-DLV-050 (표시 규칙 위임)
> ※ 미정의 항목: 가공 자재(코팅·라미넥스·박 등) 중량 가산 정책 — [NEED-CONFIRM]. 현재는 인쇄 용지 중량만 산정 대상.
