A

임직원 현황

조직도 + 사원 카드 갤러리 · 조회 중심 화면
A-1

사람 찾기

일반 임직원

특정 직원의 소속·연락처·직위를 빠르게 확인하고 싶을 때.

📍 인사 › 인사 관리 › 임직원 현황
처리 순서
  1. 상단 검색창에 성명 또는 사원번호를 입력합니다.
  2. 또는 좌측 조직도 트리에서 부서를 선택하면 해당 부서 카드 영역으로 이동합니다.
  3. 찾는 사원 카드를 클릭하면 인사정보카드가 열립니다. (→ A-5)
💡 팁 — 검색은 성명·사번 둘 다 인식합니다. 자주 찾는 사람은 카드의 ★ 로 즐겨찾기 해두세요. (→ A-4)
A-2

부서별 현황 보기

일반 임직원

본부·팀별로 누가 소속돼 있는지, 인원이 몇 명인지 훑어볼 때.

📍 임직원 현황 › 좌측 조직도
처리 순서
  1. 좌측 조직도 트리에서 회사 › 본부 › 팀/파트를 펼치거나 접습니다.
  2. 각 부서 노드에 소속 인원수가 표시됩니다.
  3. 부서를 클릭하면 우측이 해당 부서 카드 묶음으로 스크롤됩니다.
💡 팁 — 비활성 부서는 비활성 표시가 붙습니다.

A-4

연락처 바로 쓰기 · 즐겨찾기

일반 임직원

전화·메일을 바로 걸거나, 자주 찾는 사람을 상단에 고정할 때.

📍 임직원 현황 › 사원 카드
처리 순서
  1. 카드의 전화번호를 누르면 전화 앱, 이메일을 누르면 메일 작성이 열립니다.
  2. 카드의 ★ 를 누르면 즐겨찾기에 등록됩니다.
  3. 상단 즐겨찾기만 보기 · 도급직만 보기 토글로 목록을 좁힐 수 있습니다.
A-5

인사카드 열람 · PDF 출력

일반 임직원 인사 실무자

한 사람의 인사 정보 전체(신상·학력·경력·계약·서류 등)를 볼 때.

📍 임직원 현황 › 카드 클릭 → 인사정보카드
인사카드 탭 구성
  1. 기본 정보 — 신상·학력·경력·자격·어학 등
  2. 인사 정보 — 소속·발령·근로/임금 계약
  3. 급여 정보 — 급여·임금변동 등 확인 필요 (권한별 노출)
  4. 이력·현황 — 발령·평가·근태·포상징계 이력
  5. 서류 보관함 — 입사서류·증빙
PDF 출력
  1. 인사카드 우측 상단 🖨 PDF 출력 버튼을 누릅니다.
  2. 새 창이 열리며 인쇄 대화상자가 뜹니다.
  3. 대상을 "PDF로 저장" 으로 선택하면 인사기록카드가 저장됩니다.
🔒 권한 주의 — 주민등록번호·급여·평가 등 비공개정보는 열람 권한이 있는 사용자에게만 보이며, 열람 이력이 기록됩니다. 확인 필요 (역할별 노출 범위)

B-1

신규 입사자 등록

인사 실무자

신규 입사자가 확정되어 시스템에 등록할 때.

📍 임직원 관리 › 툴바 + 임직원 등록
처리 순서
  1. 툴바의 + 임직원 등록 을 클릭합니다.
  2. 기본 정보를 입력합니다 (아래 표 참고). 사번은 자동 부여됩니다.
  3. 필요 시 선택 정보(부서·직위·근로유형 등)를 펼쳐 입력합니다.
  4. 등록 을 누르면 목록에 추가되고, 온보딩이 시작됩니다. (→ B-2)

B-2

입사 온보딩 진행

인사 실무자

등록된 입사자가 계정·정보·계약·서류를 순서대로 완료하도록 진행할 때.

📍 임직원 관리 › 목록 상태 컬럼 · 인사정보카드
온보딩 7단계 확인 필요 (순서)
  1. 이메일 발송완료 — 입사자에게 안내 메일 발송
  2. 계정등록 완료 — 입사자가 계정 생성
  3. 정보등록 완료 — 입사자가 본인 정보 입력
  4. 계약 발송 — 근로/임금 계약서 서명요청 발송 (→ B-3)
  5. 계약 서명완료 — 입사자가 전자서명
  6. 서류 발송 — 입사서류 발송
  7. 서류 제출완료 — 입사자가 서류 제출 → 입사 확정
진행 상태 pill

완료모든 단계 완료 · 입사 확정

B-3

계약 서명요청 · 서류 발송

인사 실무자

입사자가 정보 입력을 마친 뒤, 근로/임금 계약과 입사서류를 발송할 때.

📍 임직원 관리 › 인사카드 › 인사 정보 탭
처리 순서
  1. 대상 직원의 인사정보카드를 열고 인사 정보 탭으로 이동합니다.
  2. 정보 등록이 완료되면 계약 항목에 서명 요청 버튼이 나타납니다. 클릭해 근로계약 서명요청을 발송합니다.
  3. 근로계약 서명완료 후 임금계약도 같은 방식으로 서명 요청 합니다.
  4. 입사서류는 서류 항목의 전체 발송 으로 일괄 발송합니다.
💡 새 근로계약서 작성 자체는 작성 버튼을 누르면 계약 관리 메뉴로 이동해 진행합니다. 계약서 상세 작성은 계약 관리 매뉴얼을 참고하세요.
⚠️ 서명 요청 버튼은 등록완료 & 미서명 상태일 때만 보입니다. 확인 필요 (표시 조건)

B-5

이메일 · 문자 발송

인사 실무자

온보딩 안내 메일을 (재)발송하거나, 대상자에게 문자를 보낼 때.

📍 임직원 관리 › 행 선택 → 툴바 이메일/문자 발송
처리 순서
  1. 목록에서 대상 행의 체크박스를 선택합니다.
  2. 툴바의 이메일 발송 또는 문자 발송 을 누릅니다.
  3. 내용을 확인하고 발송합니다. 계정 미등록 행은 개별 이메일 발송/재발송 버튼으로도 가능합니다.
⚠️ 재발송 쿨다운 — 오늘 이미 발송된 대상은 하루 뒤 재발송할 수 있습니다.

인사 실무자

  2. 부서 추가 / 이름변경 / 이동 / 구분변경 / 비활성 / 삭제 를 수행합니다.

💡 예전 「조직 관리」 메뉴는 이 부서 관리 모달로 통합되었습니다. 부서 관리 진입은 임직원 관리 화면에서 합니다.

C-1

포상 등록

인사 실무자

우수사원 포상 등 포상 건을 등록할 때. 여러 명에게 동일 내용을 한 번에 등록할 수 있습니다.

📍 포상·징계 › 툴바 + 신규 등록
처리 순서
  1. 툴바의 + 신규 등록 을 클릭합니다.
  2. 구분을 포상 으로 선택합니다. 번호(PRZ-연도-####)는 자동 부여됩니다.
  3. + 직원 선택 으로 대상자를 여러 명 선택할 수 있습니다 (N명에게 동일 내용 N건 등록).
  4. 통보일 · 결과 · 사유 를 입력하고 등록 합니다.

💡 등록한 포상 건은 대상자 인사카드의 포상·징계 이력에 자동 반영됩니다. (→ C-3)
C-2

징계 등록

인사 실무자

시말서·경고 등 징계 건을 등록할 때.

📍 포상·징계 › 툴바 + 신규 등록
처리 순서
  1. + 신규 등록 → 구분을 징계 로 선택합니다. 번호는 DSC-연도-#### 로 자동 부여됩니다.
  2. 대상 직원 · 통보일 · 결과(시말서/경고/해고) · 사유를 입력합니다.
  3. 등록 합니다. 징계는 결재선 승인 절차를 거칩니다. 확인 필요
🔒 결재선 — 징계 건의 승인 라인(누가 승인하는지)은 확인 필요 입니다. 실제 결재 흐름을 확인해 채워주세요.
C-3

포상 · 징계 조회

인사 실무자

기존 포상·징계 이력을 찾아볼 때.

📍 포상·징계 › 검색 패널
조회 방법
  1. 기간(통보일 기준, 기본 최근 1년) · 성명/사번/사유 로 검색합니다.
  2. 상세검색의 구분 필터로 포상만 / 징계만 볼 수 있습니다.
  3. 목록에서 성명을 누르면 해당 직원 인사카드가 열립니다.
⚠️ 이 화면에는 상단 포상/징계 탭이 없습니다. 두 종류가 한 목록에 구분 컬럼(🏆/⚠️)으로 섞여 표시되며, 구분은 검색 필터로 나눕니다.
C-4

수정 · 삭제

인사 실무자

등록한 포상·징계 건을 고치거나 지울 때.

📍 포상·징계 › 행 수정 버튼
처리 순서
  1. 목록에서 해당 행 우측의 수정 버튼을 누릅니다. (행 자체 클릭이 아닌 수정 버튼)
  2. 통보일·결과·사유를 고친 뒤 저장 합니다.
  3. 삭제하려면 수정 모달 안의 삭제 버튼을 사용합니다.
⚠️ 수정 시 대상 직원은 변경할 수 없습니다. 대상이 잘못됐다면 삭제 후 다시 등록하세요.
